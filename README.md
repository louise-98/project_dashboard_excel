# Projeto Dashboard de Vendas do XBOX com Excel

O projeto consiste em criar um Dashboard de Vendas/Assinaturas do Xbox, consolidando informações sobre assinantes, planos, benefícios extras (EA Play, Season Pass, Minecraft), valores pagos e métricas importantes para gestão.

O arquivo contém 4 abas principais:

Assets – Ícones e elementos gráficos padronizados.

Bases – Base de dados dos assinantes (dados brutos).

Cálculos – Tabelas auxiliares (dinâmicas) e métricas calculadas.

Dashboard – Página final com gráficos, KPIs e layout.

---

# Dados Utilizados

- Identificação: Subscriber ID; Name

- Plano: Plan (Ultimate, Standard, Core); Start Date; Auto Renewal (Yes/No); Subscription Price; Subscription Type (Monthly, Annual, Quarterly)

- Serviços Adicionais: EA Play Season Pass; EA Play Season Pass Price; Minecraft Season Pass; Minecraft Season Pass Price

- Cupons / Descontos: Coupon Value; Total Value

---

# Passos Gerais: Reprodução do Dashboard

1º) Organize e padronize os dados dos assinantes na aba "Bases".

2º) Crie tabelas auxiliares e métricas na aba "Cálculos" através de tabelas dinâmicas, respondendo as questões encontradas na aba.

3º) Monte os gráficos e indicadores na aba "Dashboard", conectando-os às tabelas da aba "Cálculos".

4º) Aplique filtros (segmentações de dados) e finalize o design com base nos ícones e padrões estabelecidos pela aba "Assets".
